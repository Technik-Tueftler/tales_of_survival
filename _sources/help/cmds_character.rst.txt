Subcommand character
==========================
All subcommands related to character are listed and described here.

.. note::
    A select view is created with this commands. Please note :ref:`limitations_for_selects` 
    for restrictions on display.

Overview
-----------------

.. list-table::
   :header-rows: 1
   :widths: 30 70

   * - Command
     - Description
   * - ``show``
     - Subcommand to show all available characters and display one after selection. This can be helpful to get an overview of the existing characters and their attributes. 
   * - ``own``
     - Subcommand to show all characters you use in games and display one after selection.
   * - ``select``
     - subcommand to select a character in a game to which you have been invited.
